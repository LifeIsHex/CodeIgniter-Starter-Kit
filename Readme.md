# CodeIgniter 3.1.13 Starter kit #


## Libraries ##

  * Drag and drop support.
  * Explorer shell context menu.
  * Unicode support.
  * AES 256 EAX encryption\decryption algorithm.
  * Secure delete source file.
  * Portable and light.
